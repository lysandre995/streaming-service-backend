# Streaming Service Backend
## Description
A dummy NestJS project for creating the endpoints for a didactic AngularJS Streaming Platform application example

## Installation

```bash
$ npm install
```

## Running the app
In order to run the application you must to initialize the database, through:
```bash
$ npm run init-database
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
